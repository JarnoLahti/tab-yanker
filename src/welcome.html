Tab Yanker Installed

Check your shortcuts.

Open Brave shortcuts, confirm following shortcuts, then start yanking tabs.

Yank current tab Alt + Shift + Y
Un-yank current tab Alt + Shift + U

If Brave does not assign the shortcuts automatically, set them manually in brave://extensions/shortcuts.

How to use

  • Press Alt+Shift+Y to yank the active tab into a new window.
  • Press Alt+Shift+U to un-yank it back or choose a target window.
  • If only one valid target window exists, Tab Yanker moves the tab automatically.
